如何用代码求根号n

求根号n的方法有很多，比如二分法、牛顿法。其中二分法是线性收敛的，大概是说随着迭代次数增加，有效数字线性地增加。而牛顿法是平方收敛的，大概是说每迭代一次，有效数字会成倍增长。但是这两个方法都有一个问题，就是初值怎么选。可能有人会说，n大于1时，根号n一定小于n，初值选n不就行了吗？在比较小的范围内确实没什么问题，但是当n很大，比如1e300的时候，你会发现开始的时候收敛非常慢，迭代很多次都不出结果。不过我们发现，计算机内存储一个实数的方式一般都是用符号+指数+底数的形式，而根号n的值，大概就是原始值的指数部分除以2。选取这个为初始值，很快就能算出结果。


const (
	mask     = 0x7FF
	shift    = 64 - 11 - 1
	bias     = 1023
	fracMask = 1<<shift - 1
)

func newton(x float64) float64 {
	// 获取内部存储格式，也就是本来是float格式的那块内存强制当int看
	ix := math.Float64bits(x)
	// 右移取出指数位，-bias是因为指数本来就加上这个值，要减回去。再右移1也就是除以2
	exp := (((ix >> shift) & mask) - bias) >> 1
	// 重新把指数和底数拼起来
	b := ((exp + bias) << shift) | (ix & fracMask)
	// 拼完当float看，这时已经求出初始值
	ans := math.Float64frombits(b)
	// 以下是牛顿法
	for i := 0; i < 8; i++ {
		ans = (ans + x/ans) / 2
	}
	return ans
}
